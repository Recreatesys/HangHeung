Theme Style

We have made 4 different beautiful theme style, You can select and use as per your requirement. You can easily customize that particular theme style, Normally user has its brand colors so the user can customize as per brand colors.

Colors

Normally, users have a logo or brand color, allowing them to customize primary colors and secondary colors according to their brand. In the fourth style, you can also use gradient colors alongside these options.

Button Style

You can set a button style with 4 different styles of the button like a leaf, square corner, light rounded corner, gradient rounded corner.

Box Style
